This repository serves as a centralized deployment framework for multiple modules:

## 🎮 Featured Game: BUBBLE POP SUPERDRIVE

The crown jewel of the **Neo.fun** matrix. A high-reflex, fast-paced arcade game built completely on an HTML5 Canvas engine.

### 🔥 Key Engineering Upgrades:
* **360° Omnidirectional Vectors:** Bubbles spawn randomly from all four screen boundaries (Top, Bottom, Left, Right) traveling along dynamic diagonal axes.
* **Live Difficulty Switcher:** Seamlessly shift between modes instantly without refreshing the canvas page:
    * 🟢 **EASY:** Relaxed pacing with infinite recycling bounds.
    * 🔵 **NORMAL:** Standard speed config for optimal practice.
    * 🔴 **HARD (Survival Mode):** Aggressive speeds. Missing even **one single bubble** causes a neural matrix breakdown (**Instant Game Over**!).
* **Anti-Freeze Matrix Core:** Features clean interval resets (`clearInterval` + `cancelAnimationFrame`) ensuring zero-freeze smooth transitions during rapid difficulty adjustments.

## 🕹️ Production Controls & Deployment

### Local Setup
To run the entire hub or test components locally:
1. Clone the repository: `git clone https://github.com/gulshanbhardwaj1/Neo.fun.git`
2. Open `index.html` in any modern web browser.

### Modifying Code Securely
* To tweak layout designs, modify the main styling stylesheets.
* To upgrade game physics or speeds, open `bubble-pop.js` and alter the values inside the `CONFIG` data matrix.
